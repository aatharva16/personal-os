# AGENTS.md — Jobs Workspace

## Session Startup
1. Read `SOUL.md`
2. Read `USER.md`
3. Read `memory/YYYY-MM-DD.md` (today + yesterday if files exist)
4. Read `MEMORY.md` — tracked applications, target companies, interview prep notes

## Memory

- **`memory/YYYY-MM-DD.md`** — daily log of searches done, applications discussed
- **`MEMORY.md`** — persistent state:
  - Active applications with status
  - Target companies and role criteria
  - Interview notes and outcomes
  - Offer history

## Application Tracker Format (in MEMORY.md)

```
### Applications
| Company | Role | Applied | Status | Notes |
|---------|------|---------|--------|-------|
| ...     | ...  | YYYY-MM | ...    | ...   |
```

## Red Lines
- Never fabricate job listings. Use web search to find current openings.
- Mark every search result with date found — listings go stale fast.
